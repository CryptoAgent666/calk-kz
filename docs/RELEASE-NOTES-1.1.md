# Release notes — v1.1 (build 6)

Для App Store Connect и Google Play Console. Поля «Что нового» / «What's New».
Версия: **iOS 1.1 (build 6)**, **Android 1.1 (versionCode 6)**, appId `kz.calk.app`.

## 🇷🇺 Русский (основной)

```
Что нового в версии 1.1:

• Все расчёты актуализированы под нормы Казахстана на 2026 год: МРП 4 325 ₸, МЗП 85 000 ₸, базовый вычет по ИПН 30 МРП, упрощёнка для ИП — 4%.
• Исправлены формулы в десятках калькуляторов: налог на имущество (прогрессивная маржинальная шкала), госпошлина за развод, пособие по безработице, микрозаймы, налог при продаже недвижимости, транспортный налог и другие.
• Калькуляторы дополнены примерами расчётов, ответами на частые вопросы и ссылками на официальные источники (adilet.zan.kz, eGov, КГД, ЕНПФ).
• Расчёты теперь обновляются автоматически — приложение всегда считает по действующему законодательству, без ожидания обновлений в сторе.
• Ускорена работа, улучшена доступность, исправлены мелкие ошибки.
```

## Примечания для подачи

- **Версия должна быть выше текущей в сторе.** Проверь в App Store Connect / Play Console актуальные version / versionCode и при необходимости подними выше 1.1 / 6.
- **Этот релиз обязателен** — он впервые включает механизм OTA-обновлений (`@capgo/capacitor-updater`). После него правки формул будут доходить в приложение без новых релизов (`npm run publish:app`).
- **Android подпись:** собирай release AAB своим upload-ключом (Play App Signing), иначе Google отклонит.
- **Скриншоты/описание** в сторе менять не обязательно — изменения внутри расчётов.
